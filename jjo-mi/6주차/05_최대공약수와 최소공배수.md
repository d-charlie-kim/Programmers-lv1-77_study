## 🔎_05. 최대공약수와 최소공배수

<b>문제 설명</b>
</br>
> 두 수를 입력받아 두 수의 최대공약수와 최소공배수를 반환하는 함수, solution을 완성해 보세요. 배열의 맨 앞에 최대공약수, 그다음 최소공배수를 넣어 반환하면 됩니다. 예를 들어 두 수 3, 12의 최대공약수는 3, 최소공배수는 12이므로 solution(3, 12)는 [3, 12]를 반환해야 합니다.
> </br>

<b>제한 조건</b>
>- 두 수는 1이상 1000000이하의 자연수입니다.

### ✍️ _풀이_

```js
function solution(n, m) {
    const gcd = (n, m) => n % m === 0 ? m : gcd(m, n % m);
    const lcm = (n, m) => n * m / gcd(n, m)
    return [gcd(n, m), lcm(n,m)];
}
```


### ✨느낀점
> 최대공배수, 공약수를 이번에 알게된 "유클리드호제법"으로 풀었음. 원리를 외워서 다른거에도 적용하면 좋을거같다. 우선 한번본걸로는 이해하기 힘들거같다
> 그래서, 배열안에 배열들을 이미 완성시켜서 바로 푸쉬해서 쌓기로함<br/>

> 다른 사람 풀이를 보니 `map`으로 한번돌면서 해당자리값을 다른 배열 자리값으로 해당값을 가져와서 기존 배열에 더해서 바로 리턴하는 형태...깔끔한 접근방식에 놀람. `map` `idx`를 잘 활용해야겠다는 생각이 듦  
```js
function sumMatrix(A,B){
    var answer;
  answer = A.map((a, i) => {
    return a.map((val, idx) => {
        val += B[i][idx];
      return val;
    })
  })

    return answer;
}

```
